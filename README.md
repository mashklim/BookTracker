# Book Tracker

## Автор
- Имя и фамилия: Машкова Мария

## Описание
GUI-приложение на Python Tkinter для ведения списка прочитанных книг.

Функции:
- Добавление книги (название, автор, жанр, страницы)
- Валидация ввода
- Фильтрация по жанру
- Фильтрация по количеству страниц (больше 200)
- Удаление выбранной книги
- Сохранение и загрузка данных из `books.json`

## Запуск
1. Откройте терминал в папке проекта:
   ```bash
   cd BookTracker
   ```
2. Запустите:
   ```bash
   python3 main.py
   ```

## Пример использования
1. Заполните поля и нажмите `Добавить книгу`.
2. Выберите жанр в блоке фильтрации и/или включите фильтр `> 200 страниц`.
3. Нажмите `Сохранить в JSON`.
4. Для восстановления данных нажмите `Загрузить из JSON`.
